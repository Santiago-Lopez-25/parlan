# The Parlan Programming Language

Parlan is an open source programming language created for learn how compilers work

## Overview
Parlan is a programming language created to help begginers to learn how does
compilers work

### Internal Pipeline
1. **Lexical Analysis:** Scans source text into tokens.
2. **Parsing:** generates the Abstract Syntax Tree from tokens.
3. **Codegen:** generates C code from AST
   
If you want to learn how compiler work and how is one implemented, feel free to 
read the comments in every source file.
